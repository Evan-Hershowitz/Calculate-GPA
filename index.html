ⓘ LSAC GPA note

LSAC recalculates GPA using all attempted undergraduate coursework and their own grade conversion table. A+ grades are counted as 4.0 (not 4.33). Withdrawals, repeated courses, and graduate credits are handled differently. This calculator uses the standard 4.0 scale.
